### Análise do Projeto

A análise técnica do projeto foi desenvolvida no MIRO, utilizando a técnica de Storytelling das funcionalidades. 
É possível observar a análise dos seguintes pontos de vista: 

- User Story
- Regras de Negocio
- Cenário de Teste
- Requisições HTTP
